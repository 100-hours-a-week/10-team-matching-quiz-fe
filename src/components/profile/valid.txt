
- 이름

입력의 길이가 부적합할 때 : “이름은 한글 최소 2자 최대 50자까지 입력 가능합니다.”
한글외의 문자가 있을 때 : “이름은 한글만 입력 가능합니다.”

- 닉네임

입력의 길이가 부적합할 때 : “닉네임은 최소 2자 최대 50자까지 입력 가능합니다.”
닉네임 형식이 부적합할 때 : “닉네임은 (영어 이름) .  (영어 성)의 형식으로 작성해주세요. 예) joy.lee”

- 과정명 
교육 과정을 선택하지 않았을 때(기본값일 때) : “교육 과정을 선택해주세요.”

- 희망 직무
선택하지 않았을 때 : 최소 1개 이상 선택해주세요.


- 기술 스택
선택하지 않았을 때 : 최소 1개 이상 선택해주세요.

- 프로필 사진
허용되지 않은 확장자의 파일 업로드 : “프로필 사진은 jpg, jpeg, png만 업로드가 가능합니다.”
허용된 용량 초과의 파일 업로드 : “프로필 사진은 5mb 이하의 파일만 업로드가 가능합니다.”

- 자리
선택하지 않았을 때 : 본인의 자리를 선택해주세요.
